# 📄 YongPDF_page (용PDF 페이지 에디터)

**YongPDF_page**는 PDF 문서의 페이지를 쉽고 직관적으로 관리할 수 있는 전문 도구입니다. 
페이지 순서 변경, 회전, 추가, 삭제, 추출, 압축 등 문서 구조 편집에 최적화되어 있으며, 썸네일과 드래그앤드롭 및 우클릭과 같이 편리하고 직관적인 UI/UX를 제공합니다.

---

## ✨ 핵심 기능

### 1. "딸깍" 직관적인 페이지 관리
*   **썸네일 뷰**: 문서 전체 구조를 한눈에 파악할 수 있는 썸네일 미리보기를 제공하며, 자유로운 크기 조절(창 크기, 페이지 크기)이 가능합니다.
*   **드래그 앤 드롭 순서 변경**: 썸네일 뷰에서 페이지를 마우스로 끌어다 놓는 것만으로 손쉽게 순서를 바꿀 수 있습니다. (페이지 추가도 가능)
*   **다중 선택 지원**: 썸네일 뷰에서 여러 페이지를 한 번에 선택하여 "우클릭"으로 삭제, 이동, 회전할 수 있습니다.

### 2. 필수 편집 도구
*   **페이지 삭제**: 불필요한 페이지를 제거할 수 있습니다.
*   **페이지 회전**: 페이지 방향을 왼쪽/오른쪽으로 90도씩 회전시킬 수 있습니다.
*   **페이지 추가**: 외부 PDF 파일을 드래그하여 현재 문서의 원하는 위치에 바로 삽입할 수 있습니다.
*   **페이지 저장**: 원하는 페이지만 골라서 별도의 PDF 파일로 저장(추출)하는 기능을 제공합니다.

### 3. 고급 기능
*   **PDF 압축**: Ghostscript를 활용한 강력한 PDF 압축 기능을 제공하여 파일 용량을 최적화할 수 있습니다. (일반/고급 모드 지원)
*   **YongPDF_text 연동**: 현재 페이지 내용을 수정하고 싶을 때 **YongPDF_text**를 바로 실행하여 텍스트를 수정할 수 있습니다.
*   **인쇄 지원**: 편집된 문서를 바로 프린터로 출력할 수 있습니다.

### 4. 사용자 편의성
*   **다크 모드 지원**: 눈이 편안한 다크 테마와 깔끔한 라이트 테마를 제공합니다.
*   **다국어 지원**: 한국어, 영어, 일본어, 중국어(간체/번체) 등 다양한 언어 환경을 지원합니다.
*   **단축키 지원**: 키보드 단축키를 통해 빠른 작업이 가능합니다.

  📄 YongPDF_text (용PDF 텍스트 에디터)

  YongPDF_text는 PDF 문서 내의 텍스트를 원본과 거의 이질감 없이 정밀하게 수정하고 재배치할 수 있는 전문 PDF 텍스트 편집 도구입니다. 
  특히 아래아한글(HWP) 문서에서 변환된 PDF의 특수성을 지원하며, 원본의 서체와 크기 자동인식, 배경색과 동일한 패치 생성 및 고화소 플래튼 기능을 통해 수정 흔적 없는 깔끔한 결과물을 선사합니다.


  ---

  ✨ 핵심 기능

  1. 직관적인 사용성
   * 쉬운 "더블 클릭" 편집: 텍스트를 "더블 클릭"하면 바로 편집창이 열립니다.
   * 원본 속성 자동인식: 원본 텍스트의 서체와 크기를 검출, 자동 적용합니다. (원본과 같은 서체가 없을 경우 설치 안내 메시지 표시됨) 
   * 패치 모드 / 지우개 모드: 원하는 위치에 패치와 텍스트를 바로 그릴 수 있고(패치 모드), 패치만 그려서 텍스트를 "지우는" 효과(지우개 모드)를 줄 수 있습니다.

  2. 고급 편집 도구
   * HWP(아래아한글)처럼 텍스트의 장평, 자간을 세밀하게 조정하여, 원본과 "거의" 같은 수정 결과물을 제공합니다.
   * 이질감 없는 수정력: 마치 배경색과 똑같은 "수정테이프"를 칠하여 수정하는 것처럼, 배경색을 자동 검출하여 패치를 씌우고 그 위에 텍스트를 그립니다. 
   * 확대 지원: 최대 800%까지 확대가 가능하여 미세한 오차도 놓치지 않고 수정할 수 있습니다.
   * 세션 관리: 작업 중인 모든 레이어 상태를 .pdfses 파일로 저장하고 나중에 다시 불러와 편집을 이어갈 수 있습니다.

  3. 정밀한 편집 경험
   * 세밀한 볼드 조절: 폰트 파일에 볼드체가 없어도 합성 볼드를 적용, 100%에서 500%까지 1% 단위로 두께를 설정할 수 있으며 이질감이 없는 자연스러운 볼드 처리를 구현합니다.
   * HWP(아래아한글) 전용 공백 보정: HWP에서 변환된 PDF 특유의 넓은 공백(한글 1자의 50%)을 감지하고 보정하여 문맥의 흐름과 자간을 완벽하게 유지합니다.
   * 정밀 위치 조정: 선택된 텍스트 오버레이를 키보드 방향키로 픽셀 단위로 미세하게 이동할 수 있습니다.

  4. 완벽한 WYSIWYG 및 고화질 저장
   * 600 DPI 플래튼: 수정된 내용을 PDF에 병합(Flatten)할 때 600 DPI의 고해상도 렌더링을 적용하여 인쇄 시에도 깨짐 없는 품질을 제공합니다.
   * 정합성 보장: 편집 화면에서 보는 텍스트의 위치, 굵기, 자간이 저장된 PDF 결과물과 높은 수준으로 정밀하게 일치합니다.
